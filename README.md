--------
[Node](https://nodejs.org/en/)
------

## Starting the client web server
> cd client
>
> npm install
>
> npm run dev
>
> open browser to http://localhost:8080

## Starting the GraphQL server
> cd server
>
> npm install
>
> npm run dev
>
> open browser to http://localhost:9000
